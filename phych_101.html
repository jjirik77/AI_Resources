03 Information Processing Lab

Memory Systems

1. Sensory Memory (The Bouncer)

Acts as gatekeeper. High capacity but extremely short duration (1-2 seconds). If you don't pay attention, it's gone.

2. Short-Term Memory (The Waiting Room)

Active conscious thought. Holds **7 +/- 2 items** (George Miller). Duration: ~20 seconds without rehearsal.

3. Long-Term Memory (The Library)

Permanent storage. Retrieval depends on "search terms" or cues. Infinite capacity.

Loftus Lab: Mandela Effect
